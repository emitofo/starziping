星海子平软件说明
一、软件介绍：
当前八字和紫微斗术门派众多，初学者常陷入不同门派排盘不同的泥沼，无法自拔。星海子平是作者经过多年学习检验，去伪存真，将经验实践验证过的技法规则保留下来，让广大易学爱好者少走弯路的一款app。同时本软件兼容不同的门派技法，提供方便灵活的不同排盘习惯选择。
二、软件功能：
1、八字排盘
八字排盘以子平法为基础，以正五行生剋为排盘依据，兼顾盲派、古法论命特点，提供纳音和部分经过验证的神煞展示，支持流年、流月、流日、流时排盘。
2、紫微斗术
紫微斗术以象数心学为依据，科权禄忌四化为基础，15颗主星为显示主体，提供紫微排盘和飞宫功能，支持大运、流年排盘。
3、六爻
六爻理论相对成熟固定，且争议较少，星海子平以《增删卜易》、《卜筮正宗》为依据，占卜解盘。
